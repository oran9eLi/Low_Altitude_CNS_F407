# V1.0 功能边界

## 必须实现

| 功能 | 说明 | 验收口径 |
|---|---|---|
| 上电自检 | 检测 HMI、GNSS、传感器、SD、LoRa、电源 | HMI/串口显示正常或异常 |
| GNSS 导航 | 解析 NMEA 数据 | 显示经纬度、高度、速度、时间 |
| 传感器采集 | 采集 MPU6050、BME280 | 显示姿态、温湿度、气压 |
| LoRa 通信 | 主控箱与外置远端节点通信 | 双向收发、心跳、ACK、丢包统计 |
| SD 日志 | CSV 数据记录 | 文件可导出，字段完整 |
| 报警 | 低电量、通信中断、模块离线 | 蜂鸣器和页面提示 |
| 电机模拟 | 4 路低速电机 PWM | 默认关闭，可控调速，急停有效 |

## 暂不强制实现

| 功能 | V1.0 处理 |
|---|---|
| 真实飞行控制 | 不做 |
| 5G 高清图传处理 | 不由 STM32 处理 |
| Remote ID 外场合规 | 预留接口，后续验证 |
| 工业远距数传完整接入 | 作为扩展 |
| 箱内 Linux + Qt | 作为 STM32MP157 升级路线 |
