# PC Qt 调试串口协议表（非 V1.0 验收项）

> 适用于 PC Qt 上位机与 STM32 主控之间的串口调试或后续上位机预研。
> V1.0 正式本地人机界面以 DWIN 串口 HMI 为准，本表不作为 V1.0 固件验收项。

## 建议帧格式

```text
帧头  命令  长度  数据区  校验  帧尾
0xAA  CMD   LEN   DATA    CRC   0x55
```

| 字段 | 长度 | 说明 |
|---|---:|---|
| 帧头 | 1 | 固定 0xAA |
| 命令 | 1 | 命令字 |
| 长度 | 1 或 2 | 数据区长度 |
| 数据区 | N | 业务数据 |
| 校验 | 1 或 2 | CRC8/CRC16，待定 |
| 帧尾 | 1 | 固定 0x55 |

## 命令字

| CMD | 名称 | 方向 | 说明 | 状态 |
|---|---|---|---|---|
| 0x01 | HEARTBEAT | 双向 | 心跳 | 待确认 |
| 0x10 | GET_STATE | Qt -> STM32 | 请求系统状态 | 待确认 |
| 0x11 | STATE_REPORT | STM32 -> Qt | 上报系统状态 | 待确认 |
| 0x20 | SET_PARAM | Qt -> STM32 | 设置参数 | 待确认 |
| 0x30 | GET_LOG_STATUS | Qt -> STM32 | 查询日志状态 | 待确认 |
| 0x40 | MOTOR_CTRL | Qt -> STM32 | 电机控制命令 | 待确认 |
