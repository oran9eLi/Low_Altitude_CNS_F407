# LoRa 通信协议表

> 用于主控箱 LoRa 主端与外置远端节点之间的点对点通信。

## 远端节点定义

远端节点不放入主控箱，作为外置通信对端。

V1.0 可选形式：

1. `E22 LoRa + USB-TTL + 电脑串口助手`
2. `E22 LoRa + 小 MCU + LED/OLED + 电池`

## 建议帧格式

```text
SOF  SRC  DST  SEQ  TYPE  LEN  PAYLOAD  CRC
```

| 字段 | 说明 |
|---|---|
| SOF | 帧头，待定 |
| SRC | 源地址 |
| DST | 目标地址 |
| SEQ | 序号，用于 ACK 和丢包统计 |
| TYPE | 帧类型 |
| LEN | 数据长度 |
| PAYLOAD | 数据区 |
| CRC | 校验 |

## 帧类型

| TYPE | 名称 | 方向 | 说明 |
|---|---|---|---|
| 0x01 | HEARTBEAT | 双向 | 链路在线检测 |
| 0x02 | ACK | 双向 | 应答 |
| 0x10 | STATE | 远端 -> 主控箱 | 远端状态上报 |
| 0x20 | COMMAND | 主控箱 -> 远端 | 指令下发 |
| 0x30 | TEST_DATA | 双向 | 教学测试数据 |

## 验收指标

| 指标 | V1.0 要求 |
|---|---|
| 双向收发 | 必须支持 |
| ACK | 必须支持 |
| 丢包统计 | 必须支持 |
| RSSI | 若模块支持则显示 |
| 远端节点独立供电 | 教学版建议支持 |
