# 完整引脚分配表 (STM32F407ZGTx)

> 方案：FSMC 并口屏（ATK-MD0700 7 寸）+ 4 路电机 + SDIO + CAN1 + USART6
> 开发板：STM32F407 开发板
> **原 DWIN 串口屏方案已废弃**：PA0=WK_UP 按键（硬件固定），LoRa 无法使用 UART4，移至 USART3 后无 UART 留给 DWIN 屏，V1.0 都无法完成。
> 共 **62 引脚**

## 关键设计决策

```
  CAN1 改用 PA11/PA12（经 P5 跳线，信号干净）→ PB8/PB9 释放
  PA0 硬件固定 WK_UP → LoRa UART4 不可用 → LoRa 用 USART3 PB10/PB11
  FSMC 屏不需 UART → 释放 USART3 给 LoRa → 唯一可行方案
  电机 DIR1→PA8(DCMI_XCLK) DIR2→PE5(DCMI_D6) DIR3→PE6(DCMI_D7) DIR4→PA4(DAC_OUT1)
  蜂鸣器 PF6→PF8 TIM13_CH1 (PF6=GBC_KEY 不可用)
  电源 ADC PC3→PA5 ADC1_IN5 (PC3=I2S_SDIN ES8388)
  Touch 走 TFTLCD 插座 PB0/PB1/PF11/PC13 直连，零飞线
  电机 PWM3/4→PD12/PD13 TIM4_CH1/2 (PB0/PB1 还给 Touch)
  禁用摄像头(DCMI)→释放 PA8 PE5 PE6 PB6 PB7
  禁用音频(ES8388)→释放 PC6 PC7
  以太网 PHY 保持复位→释放 PA1 PA7 PC4 PC5 PG11 PG13 PG14
  PC0/PC1 释放为空闲 GPIO
```
