## 核心数据结构

### Sensor_Value_t (传感器值联合体)

```c
// sensor_data.h (已存在)
typedef struct {
    double latitude;      // 纬度 (度)
    double longitude;     // 经度 (度)
    float  altitude;      // 海拔 (米)
    float  speed;         // 地面速度 (km/h)
    float  heading;       // 航向 (度)
    float  hdop;          // 水平精度因子
    uint8_t sats;         // 可见卫星数
    uint8_t fix;          // 定位类型 (0=无效 1=GPS 2=DGPS)
    uint32_t utc_sec;     // UTC 时间戳
} Sensor_GeoValue_t;
```

### Sensor_Driver_t (驱动接口表)

```c
// sensor_driver.h (已存在)
typedef struct Sensor_Driver {
    Sensor_DeviceId_t id;
    Sensor_Status_t (*Init)(void);
    Sensor_Status_t (*SelfCheck)(void);
    Sensor_Status_t (*Read)(Sensor_Value_t *out);
    Sensor_Status_t (*Reset)(void);
} Sensor_Driver_t;
```

### NMEA 解析内部结构

```c
// sensor_nmea.h (新增)
#define NMEA_SENTENCE_MAX_LEN  128
#define NMEA_FIELD_MAX         20

typedef enum {
    NMEA_TYPE_GGA = 0,
    NMEA_TYPE_RMC,
    NMEA_TYPE_GSV,
    NMEA_TYPE_UNKNOWN
} Nmea_Type_t;

typedef struct {
    Nmea_Type_t type;
    char        fields[NMEA_FIELD_MAX][NMEA_SENTENCE_MAX_LEN];
    uint8_t     field_count;
    uint8_t     checksum_ok;
} Nmea_Sentence_t;
```

## 数据流

```
ATGM336H → USART2 → DMA缓冲区 → Idle中断 →
  ↓
DMA半满/全满中断 → 环形缓冲 → NMEA帧提取 ($...*\r\n)
  ↓
校验和验证 → Split fields by ',' → 分类型解析
  ↓
GGA解析: UTC/纬度/经度/质量/卫星数/HDOP/高度
RMC解析: UTC/状态/纬度/经度/速度/航向/日期
GSV解析: 卫星数/编号/仰角/方位角/信噪比
  ↓
填充 Sensor_GeoValue_t → s_gnss_data (静态缓存)
  ↓
SensorRegistry_ReadAll() → 拷贝到 App 层 Buffer
```

---

## DMA Idle 接收方案

```
配置:
  USART2_RX → DMA1_Stream5 (或 Stream6)
  缓冲区大小: 512 字节
  模式: Circular

中断:
  USART2_IRQHandler → __HAL_UART_CLEAR_IDLEFLAG
                    → HAL_UARTEx_RxEventCallback (HAL_UART_RXEVENT_IDLE)
                    → 处理环形缓冲 NMEA 帧

优势:
  - 1Hz 出帧速度快条件下无 CPU 开销
  - Idle 中断自动检测帧间隔 (UART空idle)
```

## 接口约定

| 接口 | 说明 |
|------|------|
| `BSP_GNSS_Init()` | USART2 + DMA 初始化，返回 `HAL_StatusTypeDef` |
| `SensorGnss_Init()` | 初始化解析器状态机，返回 `SENSOR_OK` / `SENSOR_ERROR` |
| `SensorGnss_SelfCheck()` | 检测 `s_gnss_data.fix>0` 且 `sats≥4`，超时 60s |
| `SensorGnss_Read()` | 拷贝当前 `s_gnss_data` 至 `out`，无阻塞 |
| `SensorGnss_Reset()` | 清空缓冲区，重置状态机 |
